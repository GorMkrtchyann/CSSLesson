CSS-ի Կառուցվածքը

CSS-ն այն լեզուն է, որը մենք օգտագործում ենք վեբ էջի ոճավորման համար:

Ի՞նչ է CSS-ը

  • CSS-ը նշանակում է Cascading Style Sheets
  • CSS-ը նկարագրում է, թե ինչպես պետք է HTML տարրերը ցուցադրվեն էկրանին
  • CSS-ը խնայում է շատ աշխատանք: Այն կարող է միանգամից վերահսկել բազմաթիվ վեբ էջերի դասավորությունը
  • Արտաքին ոճաթերթերը պահվում են CSS ֆայլերում

Ինչու՞ օգտագործել CSS:

CSS-ն օգտագործվում է ձեր վեբ էջերի ոճերը սահմանելու համար, ներառյալ դիզայնը, դասավորությունը և տարբեր սարքերի և էկրանի չափերի ցուցադրման տատանումները:

ՕՐԻՆԱԿ

body {
background-color: lightblue;
}

h1 {
color: white;
text-align: center;
}

p {
font-family: verdana;
font-size: 20px;
}

CSS Ընտրիչներ

CSS ընտրիչները օգտագործվում են «գտնելու» (կամ ընտրելու) HTML տարրերը, որոնք ցանկանում եք ոճավորել:

Մենք կարող ենք CSS ընտրիչները բաժանել հինգ կատեգորիաների

  • Պարզ ընտրիչներ (ընտրեք տարրեր՝ հիմնվելով անվան, ID-ի, class-ի վրա)
  • Կոմբինատորի ընտրիչներ (ընտրեք տարրեր՝ հիմնվելով նրանց միջև որոշակի հարաբերությունների վրա)
  • Կեղծ դասի ընտրիչներ (ընտրեք տարրեր որոշակի վիճակի հիման վրա)
  • Կեղծ տարրերի ընտրիչներ (ընտրել և ոճավորել տարրի մի մասը)
  • Հատկանիշների ընտրիչներ (ընտրեք տարրեր՝ հիմնված հատկանիշի կամ հատկանիշի արժեքի վրա)

CSS class Ընտրիչ

Class ընտրիչը ընտրում է HTML տարրեր՝ հատուկ դասի հատկանիշով:

Հատուկ class տարրեր ընտրելու համար գրեք կետ (.) նիշ, որին հաջորդում է դասի անվանումը:

ՕՐԻՆԱԿ

}

ՕՐԻՆԱԿ

}

CSS Խմբավորման Ընտրիչ

Խմբավորման ընտրիչը ընտրում է բոլոր HTML տարրերը նույն ոճի սահմանումներով:

Նայեք հետևյալ CSS կոդը (h1, h2 և p տարրերը ունեն նույն ոճի սահմանումները).

h1 {
color: red;
text-align: center;
}

h2 {
color: red;
text-align: center;
}

p {
color: red;
text-align: center;
}

Ավելի լավ կլինի սելեկտորները խմբավորել, կոդը նվազագույնի հասցնել։

Ընտրիչներ խմբավորելու համար յուրաքանչյուր ընտրիչ առանձնացրեք ստորակետով:

ՕՐԻՆԱԿ

h1, h2, p {
color: red;
text-align: center;
}

CSS background-color

background-color հատկությունը սահմանում է տարրի ֆոնի գույնը:

ՕՐԻՆԱԿ

body {
background-color: lightblue;
}

CSS-ով գույնը առավել հաճախ նշվում է հետևյալով.

  • վավեր գույնի անուն, ինչպես "red"
  • HEX արժեք, ինչպես "#ff0000"
  • RGB արժեք, ինչպես "rgb(255,0,0)"
  • RGBA արժեք, ինչպես "rgba(255,0,0, 0.6)"

Այլ տարրեր

Դուք կարող եք սահմանել ֆոնի գույնը ցանկացած HTML տարրի համար

}

}

  • dashed - Սահմանում է կտրված եզրագիծ
  • solid - Սահմանում է սովորական եզրագիծ
  • double - Սահմանում է կրկնակի եզրագիծ

  • inset - Սահմանում է 3D ներդիր եզրագիծ: Էֆեկտը կախված է եզրագծի գույնի արժեքից
  • outset - Սահմանում է 3D սկզբնական սահման: Էֆեկտը կախված է եզրագծի գույնի արժեքից
  • none - Եզրագիծ չի սահմանում
  • hidden - Սահմանում է թաքնված եզրագիծ

Լայնությունը կարող է սահմանվել որպես որոշակի չափ (px, pt, cm, em և այլն) կամ օգտագործելով երեք նախապես սահմանված արժեքներից մեկը՝ բարակ, միջին կամ հաստ.

ՕՐԻՆԱԿ

p.one {
border-style: dotted;
border-width: 5px;
}
p.two {
border-style: dotted;
border-width: medium;
}
p.tree {
border-style: solid;
border-width: 2px;
}
p.four {
border-style: solid;
border-width: thick;
}

CSS Եզրագծի Գույնը

border-color հատկությունն օգտագործվում է չորս եզրագծերի գույնը սահմանելու համար:

Գույնը կարող է սահմանվել հետևյալով.

  • անուն - նշեք գույնի անուն, ինչպես "red"
  • HEX - նշեք HEX արժեք, ինչպես "#ff0000"
  • RGB - նշեք RGB արժեք, օրինակ՝ "rgb(255,0,0)":

Նշում: Եթե սահմանի գույնը սահմանված չէ, այն ժառանգում է տարրի գույնը:

ՕՐԻՆԱԿ

p.one {
border-style: solid;
border-color: red;
}
p.two {
border-style: solid;
border-color: green;
}
p.tree {
border-style: solid;
border-color: blue;
}

Կոդը կրճատելու համար հնարավոր է նաև նշել բոլոր առանձին եզրագծաին հատկությունները մեկ սեփականության մեջ:

Լուսանցքները օգտագործվում են տարրերի շուրջ տարածություն ստեղծելու համար՝ ցանկացած եզրագծերի սահմաններից դուրս:

ՕՐԻՆԱԿ

p {

}

CSS margin հատկությունները օգտագործվում են տարրերի շուրջ տարածություն ստեղծելու համար՝ ցանկացած եզրագծերի սահմաններից դուրս:

CSS-ով դուք լիովին վերահսկում եք լուսանցքները: Գոյություն ունեն տարրի յուրաքանչյուր կողմի (վերև, աջ, ներքև և ձախ) լուսանցք սահմանելու հատկություններ:

CSS-ն ունի տարրի յուրաքանչյուր կողմի լուսանցքը նշելու հատկություններ.

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Բոլոր լուսանցքային հատկությունները կարող են ունենալ հետևյալ արժեքները

  • ավտոմատ - զննարկիչը հաշվարկում է լուսանցքը
  • երկարություն - նշում է margin px, pt, cm և այլն:
  • % - սահմանում է margin պարունակող տարրի լայնության %-ով
  • ժառանգել - նշում է, որ սահմանը պետք է ժառանգվի մայր տարրից

Հուշում. Բացասական արժեքները թույլատրվում են:

CSS height և widht արժեքներ

height և widht հատկությունները կարող են ունենալ հետևյալ արժեքները.

  • auto - Սա լռելյայն(default) է: Բրաուզերը հաշվարկում է բարձրությունը և լայնությունը
  • երկարություն - Սահմանում է բարձրությունը/լայնությունը px-ով, սմ-ով և այլն:
  • % - Սահմանում է բարձրությունը/լայնությունը պարունակող բլոկի տոկոսներով
  • initial - Բարձրությունը/լայնությունը սահմանում է իր լռելյայն արժեքին
  • inherit - Բարձրությունը/լայնությունը կժառանգվի իր մայր արժեքից

CSS Box մոդելը(model)

CSS-ում «box model» տերմինն օգտագործվում է դիզայնի և դասավորության մասին խոսելիս։

CSS box model-ը, ըստ էության, տուփ է, որը պարուրում է յուրաքանչյուր HTML տարրի շուրջը: Այն բաղկացած է margin-ից, border-ից, padding-ից և իրական բովանդակությունից: Ստորև բերված պատկերը ցույց է տալիս տուփի մոդելը.


Տարբեր մասերի բացատրություն

  • Բովանդակություն - տուփի բովանդակությունը, որտեղ հայտնվում են տեքստը և պատկերները
  • Padding - Մաքրում է բովանդակության շուրջ տարածքը
  • Border - եզրագիծ, որը շրջանցում է padding-ը և բովանդակությունը
  • Margin - Մաքրում է սահմանից դուրս գտնվող տարածքը:

CSS Outline ոճ

outline-style հատկությունը սահմանում է ուրվագծի ոճը և կարող է ունենալ հետևյալ արժեքներից մեկը.

  • dotted - Սահմանում է կետավոր ուրվագիծ
  • dashed - Սահմանում է կտրված ուրվագիծ
  • solid - Սահմանում է սովորական ուրվագիծ
  • double - Սահմանում է կրկնակի ուրվագիծ
  • groove - Սահմանում է 3D ակոսավոր ուրվագիծ: Էֆեկտը կախված է ուրվագծի գույնի արժեքից
  • ridge - Սահմանում է 3D շողքային ուրվագիծ: Էֆեկտը կախված է ուրվագծի գույնի արժեքից
  • inset - Սահմանում է 3D ներդիր ուրվագիծ: Էֆեկտը կախված է ուրվագծի գույնի արժեքից
  • outset - Սահմանում է 3D սկզբնական ուրվագիծ: Էֆեկտը կախված է ուրվագծի գույնի արժեքից
  • none - ՈԻրվագիծ չի սահմանում
  • hidden - Սահմանում է թաքնված ուրվագիծ

Հետևյալ օրինակը ցույց է տալիս outline-style տարբեր արժեքներ.

ՕՐԻՆԱԿ

p.dotted {outline-style: dotted;}
p.dashed {outline-style: dashed;}
p.solid {outline-style: solid;}
p.double {outline-style: double;}
p.groove {outline-style: groove;}
p.ridge {outline-style: ridge;}
p.inset {outline-style: inset;}
p.outset {outline-style: outset;}

CSS-ն ունի տեքստի ձևաչափման բազմաթիվ հատկություններ:


Տեքստի Գույն

color հատկությունն օգտագործվում է տեքստի գույնը սահմանելու համար: Գույնը նշվում է հետևյալով.

  • գույնի անուն, ինչպես "red"
  • HEX արժեք, ինչպես "#ff0000"
  • RGB արժեք, ինչպես "rgb(255,0,0)"

Էջի լռելյայն(default) տեքստի գույնը սահմանվում է մարմնի ընտրիչում:

ՕՐԻՆԱԿ

color: red;
}
.p1 {
color: blue;
}

text-align հատկությունն օգտագործվում է տեքստի հորիզոնական հավասարեցումը սահմանելու համար:

Հետևյալ օրինակը ցույց է տալիս կենտրոնացված և ձախ և աջ հավասարեցված տեքստը (ձախ հավասարեցումը կանխադրված է, եթե տեքստի ուղղությունը ձախից աջ է, իսկ աջ հավասարեցումը կանխադրված է, եթե տեքստի ուղղությունը աջից ձախ է):

ՕՐԻՆԱԿ

.p1 {
text-align: center;
}
.p2 {
text-align: left;
}
.p3 {
text-align: right;
}

text-decoration-line հատկությունն օգտագործվում է տեքստին հարդարման տող ավելացնելու համար:

Հուշում: Դուք կարող եք միավորել մեկից ավելի արժեքներ, օրինակ՝ գծի և ընդգծման՝ տեքստի վրա և տեքստի տակ տողերը ցուցադրելու համար:

ՕՐԻՆԱԿ

.p1 {
text-decoration-line: overline;
}
.p2 {
text-decoration-line: line-through;
}
.p3 {
text-decoration-line: underline;
}
.p4 {
text-decoration-line: overline underline;
}
