CSS-ի Կառուցվածքը

CSS-ն այն լեզուն է, որը մենք օգտագործում ենք վեբ էջի ոճավորման համար:

Ի՞նչ է CSS-ը

  • CSS-ը նշանակում է Cascading Style Sheets
  • CSS-ը նկարագրում է, թե ինչպես պետք է HTML տարրերը ցուցադրվեն էկրանին
  • CSS-ը խնայում է շատ աշխատանք: Այն կարող է միանգամից վերահսկել բազմաթիվ վեբ էջերի դասավորությունը
  • Արտաքին ոճաթերթերը պահվում են CSS ֆայլերում

Ինչու՞ օգտագործել CSS:

CSS-ն օգտագործվում է ձեր վեբ էջերի ոճերը սահմանելու համար, ներառյալ դիզայնը, դասավորությունը և տարբեր սարքերի և էկրանի չափերի ցուցադրման տատանումները:

ՕՐԻՆԱԿ

body {
background-color: lightblue;
}

h1 {
color: white;
text-align: center;
}

p {
font-family: verdana;
font-size: 20px;
}

HTML-ը ստեղծվել է վեբ էջի բովանդակությունը նկարագրելու համար, օրինակ՝

CSS Շարահյուսություն

CSS կանոնը բաղկացած է ընտրիչից և հայտարարագրման բլոկից:

CSS Շարահյուսություն

Ընտրիչը ցույց է տալիս HTML տարրը, որը ցանկանում եք ոճավորել:

Հայտարարությունների բլոկը պարունակում է մեկ կամ մի քանի հայտարարություններ, որոնք բաժանված են ստորակետերով:

Յուրաքանչյուր հայտարարագիր ներառում է CSS տեսքի հայտարարող կոդի անվանում և արժեք՝ առանձնացված երկու կետով:

Բազմաթիվ CSS հայտարարագրեր բաժանված են կետ-ստորակետերով, իսկ հայտարարագրման բլոկները շրջապատված են ձևավոր փակագծերով:

ՕՐԻՆԱԿ

Այս օրինակում բոլոր <p> տարրերը կհավասարեցվեն կենտրոնական մասում՝ տեքստի կարմիր գույնով.

p {
text-align: center;
color: red;
}

Բացատրված օրինակը

  • p-ն ընտրիչ է CSS-ում (այն մատնանշում է HTML տարրը, որը ցանկանում եք ոճավորել. <p>):
  • color հատկություն է, իսկ red արժեքն է
  • text-align-ը հատկություն է, իսկ center արժեքն է


CSS Ընտրիչներ

CSS ընտրիչը ընտրում է HTML տարր(եր)ը, որը ցանկանում եք ոճավորել:

CSS Ընտրիչներ

CSS ընտրիչները օգտագործվում են «գտնելու» (կամ ընտրելու) HTML տարրերը, որոնք ցանկանում եք ոճավորել:

Մենք կարող ենք CSS ընտրիչները բաժանել հինգ կատեգորիաների

  • Պարզ ընտրիչներ (ընտրեք տարրեր՝ հիմնվելով անվան, ID-ի, class-ի վրա)
  • Կոմբինատորի ընտրիչներ (ընտրեք տարրեր՝ հիմնվելով նրանց միջև որոշակի հարաբերությունների վրա)
  • Կեղծ դասի ընտրիչներ (ընտրեք տարրեր որոշակի վիճակի հիման վրա)
  • Կեղծ տարրերի ընտրիչներ (ընտրել և ոճավորել տարրի մի մասը)
  • Հատկանիշների ընտրիչներ (ընտրեք տարրեր՝ հիմնված հատկանիշի կամ հատկանիշի արժեքի վրա)

Այս էջը կբացատրի CSS-ի ամենահիմնական ընտրիչները:

CSS Տարրի Ընտրիչ

Տարրերի ընտրիչը ընտրում է HTML տարրեր՝ հիմնվելով տարրի անվան վրա:

ՕՐԻՆԱԿ

Այս օրինակում բոլոր <p> տարրերը կհավասարեցվեն կենտրոնական մասում՝ տեքստի կարմիր գույնով.

p {
text-align: center;
color: red;
}

CSS id Ընտրիչ

id ընտրիչն օգտագործում է HTML տարրի id հատկանիշը՝ կոնկրետ տարր ընտրելու համար:

Տարրի id-ն եզակի է էջի ներսում, ուստի id ընտրիչն օգտագործվում է մեկ եզակի տարր ընտրելու համար:

Հատուկ id-ով տարր ընտրելու համար գրեք հեշթեգ (#) նիշ, որին հաջորդում է տարրի id-ը:

ՕՐԻՆԱԿ

Ստորև բերված CSS կանոնը կկիրառվի HTML տարրի վրա՝ id="para1"-ով:

#para1 {
text-align: center;
color: red;
}

Նշում: ID անունը չի կարող սկսվել թվով:

CSS class Ընտրիչ

Class ընտրիչը ընտրում է HTML տարրեր՝ հատուկ դասի հատկանիշով:

Հատուկ class տարրեր ընտրելու համար գրեք կետ (.) նիշ, որին հաջորդում է դասի անվանումը:

ՕՐԻՆԱԿ

Այս օրինակում class="center" բոլոր HTML տարրերը կարմիր և կենտրոնացված կլինեն

.center {
text-align: center;
color: red;
}

Կարող եք նաև նշել, որ դասի վրա պետք է ազդի միայն կոնկրետ HTML տարրերը:

ՕՐԻՆԱԿ

Այս օրինակում միայն <p> տարրերը՝ class="center"-ով կարմիր և կենտրոնացված կլինեն:

p.center {
text-align: center;
color: red;
}

HTML տարրերը կարող են նաև վերաբերել մեկից ավելի class-ների:

ՕՐԻՆԱԿ

Այս օրինակում <p> տարրը ոճավորվելու է ըստ class="center" և class="large":

<p class="center large" >Պարագրաֆ</p>

Նշում: Class անունը չի կարող սկսվել թվով:

CSS ՈՒնիվերսալ Ընտրիչ

Ունիվերսալ ընտրիչը (*) ընտրում է էջի բոլոր HTML տարրերը:

ՕՐԻՆԱԿ

Ստորև բերված CSS կանոնը կազդի էջի յուրաքանչյուր HTML տարրի վրա.

* {
text-align: center;
color: blue;
}

CSS Խմբավորման Ընտրիչ

Խմբավորման ընտրիչը ընտրում է բոլոր HTML տարրերը նույն ոճի սահմանումներով:

Նայեք հետևյալ CSS կոդը (h1, h2 և p տարրերը ունեն նույն ոճի սահմանումները).

h1 {
color: red;
text-align: center;
}

h2 {
color: red;
text-align: center;
}

p {
color: red;
text-align: center;
}

Ավելի լավ կլինի սելեկտորները խմբավորել, կոդը նվազագույնի հասցնել։

Ընտրիչներ խմբավորելու համար յուրաքանչյուր ընտրիչ առանձնացրեք ստորակետով:

ՕՐԻՆԱԿ

h1, h2, p {
color: red;
text-align: center;
}

Բոլոր CSS Պարզ Ընտրիչները

Ընտրիչ Օրինակ Օրինակի Նկարագրություն
#id #firstname Ընտրում է id="firstname" տարրը
.class .intro Ընտրում է բոլոր տարրերը class="intro"-ով
տարր.class p.intro Ընտրում է միայն <p> տարրեր class="intro"-ով
* * Ընտրում է բոլոր տարրերը
տարր p Ընտրում է բոլոր <p> տարրերը
տարր, տարր,.. div, p Ընտրում է բոլոր <div> տարրերը և բոլոր <p> տարրերը


CSS Backgrounds (ֆոն)

CSS ֆոնային հատկությունները օգտագործվում են տարրերի համար ֆոնային էֆեկտներ ավելացնելու համար:

Այս գլուխներում դուք կիմանաք հետևյալ CSS ֆոնային հատկությունների մասին

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position
  • background (սղագրություն)

CSS background-color

background-color հատկությունը սահմանում է տարրի ֆոնի գույնը:

ՕՐԻՆԱԿ

body {
background-color: lightblue;
}

CSS-ով գույնը առավել հաճախ նշվում է հետևյալով.

  • վավեր գույնի անուն, ինչպես "red"
  • HEX արժեք, ինչպես "#ff0000"
  • RGB արժեք, ինչպես "rgb(255,0,0)"
  • RGBA արժեք, ինչպես "rgba(255,0,0, 0.6)"

Այլ տարրեր

Դուք կարող եք սահմանել ֆոնի գույնը ցանկացած HTML տարրի համար

ՕՐԻՆԱԿ

Այստեղ <h1>, <p> և <div> տարրերը կունենան տարբեր ֆոնային գույներ.

div {
background-color: lightblue;
}

h1 {
background-color: blue;
}

p {
background-color: green;
}

Անթափանցիկություն / Թափանցիկություն

opacity հատկությունը սահմանում է տարրի անթափանցիկությունը/թափանցիկությունը: Այն կարող է վերցնել 0,0 - 1,0 արժեք: Որքան ցածր է արժեքը, այնքան ավելի թափանցիկ.

opacity 1

opacity 0.6

opacity 0.3

opacity 0.1

ՕՐԻՆԱԿ

div {
background-color: blue;
opacity: 0.6;
}

Նշում: Անթափանցիկության հատկությունն օգտագործելով տարրի ֆոնին թափանցիկություն ավելացնելու համար, նրա բոլոր զավակ տարրերը ժառանգում են նույն թափանցիկությունը:

Թափանցիկություն՝ օգտագործելով RGBA

Եթե դուք չեք ցանկանում անթափանցիկություն կիրառել զավակ տարրերի վրա, ինչպես վերը նշված մեր օրինակում, օգտագործեք RGBA գույնի արժեքները: Հետևյալ օրինակը սահմանում է ֆոնի գույնի անթափանցիկությունը և ոչ թե տեքստը.

CSS եզրագծի ոճ

border-style սահմանում է, թե ինչպիսի եզրագիծ պետք է ցուցադրվի:

Թույլատրվում են հետևյալ արժեքները.

  • dotted - Սահմանում է կետավոր եզրագիծ
  • dashed - Սահմանում է կտրված եզրագիծ
  • solid - Սահմանում է սովորական եզրագիծ
  • double - Սահմանում է կրկնակի եզրագիծ
  • groove - Սահմանում է 3D ակոսավոր եզրագիծ: Էֆեկտը կախված է եզրագծի գույնի արժեքից
  • ridge - Սահմանում է 3D շողքային եզրագիծ: Էֆեկտը կախված է եզրագծի գույնի արժեքից
  • inset - Սահմանում է 3D ներդիր եզրագիծ: Էֆեկտը կախված է եզրագծի գույնի արժեքից
  • outset - Սահմանում է 3D սկզբնական սահման: Էֆեկտը կախված է եզրագծի գույնի արժեքից
  • none - Եզրագիծ չի սահմանում
  • hidden - Սահմանում է թաքնված եզրագիծ

border-style հատկությունը կարող է ունենալ մեկից չորս արժեք (վերևի եզրագծի, աջ եզրագծի, ներքևի եզրագծի և ձախ եզրագծի համար):

p.double {border-style: double;}
p.groove {border-style: groove;}
p.ridge {border-style: ridge;}
p.inset {border-style: inset;}
p.outset {border-style: outset;}
p.none {border-style: none;}
p.hidden {border-style: hidden;}

border-color հատկությունն օգտագործվում է չորս եզրագծերի գույնը սահմանելու համար:

Գույնը կարող է սահմանվել հետևյալով.

  • անուն - նշեք գույնի անուն, ինչպես "red"
  • HEX - նշեք HEX արժեք, ինչպես "#ff0000"

CSS-ով դուք լիովին վերահսկում եք լուսանցքները: Գոյություն ունեն տարրի յուրաքանչյուր կողմի (վերև, աջ, ներքև և ձախ) լուսանցք սահմանելու հատկություններ:

CSS-ն ունի տարրի յուրաքանչյուր կողմի լուսանցքը նշելու հատկություններ.

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Բոլոր լուսանցքային հատկությունները կարող են ունենալ հետևյալ արժեքները

  • ավտոմատ - զննարկիչը հաշվարկում է լուսանցքը
  • երկարություն - նշում է margin px, pt, cm և այլն:
  • % - սահմանում է margin պարունակող տարրի լայնության %-ով
  • ժառանգել - նշում է, որ սահմանը պետք է ժառանգվի մայր տարրից

Հուշում. Բացասական արժեքները թույլատրվում են:

CSS Կարգավորում height և widht

height և widht հատկությունները օգտագործվում են տարրի բարձրությունը և լայնությունը սահմանելու համար:

height և widht հատկությունները չեն ներառում padding, border(եզրագիծ) կամ margin(լուսանցք): Այն սահմանում է տարրի padding-ի, border-ի և margin-ի տարածքի բարձրությունը/լայնությունը:

CSS height և widht արժեքներ

height և widht հատկությունները կարող են ունենալ հետևյալ արժեքները.

  • auto - Սա լռելյայն(default) է: Բրաուզերը հաշվարկում է բարձրությունը և լայնությունը
  • երկարություն - Սահմանում է բարձրությունը/լայնությունը px-ով, սմ-ով և այլն:
  • % - Սահմանում է բարձրությունը/լայնությունը պարունակող բլոկի տոկոսներով
  • initial - Բարձրությունը/լայնությունը սահմանում է իր լռելյայն արժեքին
  • inherit - Բարձրությունը/լայնությունը կժառանգվի իր մայր արժեքից

CSS height և widht Օրինակներ

ՕՐԻՆԱԿ

.p1 {
width: 50%;
height: 200px;
}

Սահմանել max-widht

max-widht հատկությունն օգտագործվում է տարրի առավելագույն լայնությունը սահմանելու համար:

max-widht կարող է սահմանվել երկարության արժեքներով, օրինակ՝ px, սմ և այլն, կամ պարունակող բլոկի տոկոսներով (%), կամ սահմանել ոչ մեկը (սա լռելյայն է: Նշանակում է, որ չկա առավելագույն լայնություն):

ՕՐԻՆԱԿ

.p1 {
max-width: 500px;
height: 100px;
}

CSS-ի չափման բոլոր հատկությունները

Հատկություն Նկարագրություն
height Սահմանում է տարրի բարձրությունը
max-height Սահմանում է տարրի առավելագույն բարձրությունը
max-width Սահմանում է տարրի առավելագույն լայնությունը
min-height Սահմանում է տարրի նվազագույն բարձրությունը
min-width Սահմանում է տարրի նվազագույն լայնությունը
width Սահմանում է տարրի լայնությունը
